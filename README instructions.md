1)How to Install the RSI Cheatcode Indicators on TradingView

Welcome! This guide will walk you through the process of adding the @meyefoundation RSI cheatcode (Candlestick) and @meyefoundation RSI cheatcode (Trendline) indicators to your TradingView charts.

Even if you have never used code before, you can complete this process in less than 5 minutes.

---

 Prerequisites

Before you begin, make sure you have:

1. A free or paid account on [TradingView](https://www.tradingview.com/).
2. This GitHub repository open so you can copy the code.

---

 Step-by-Step Installation Guide

 Step 1: Open Your TradingView Chart

1. Go to [TradingView.com](https://www.tradingview.com/) and log into your account.
2. Click on Product at the top of the page, then select Supercharts (or simply open any asset chart you normally look at).

 Step 2: Open the Pine Editor

1. Look at the very bottom of your chart screen. You will see a row of tabs (such as Stock Screener, Pine Editor, Strategy Tester, Trading Panel).
2. Click on the tab named Pine Editor.
3. This will open a panel at the bottom of your screen where code can be pasted.

> Note: If the Pine Editor isn't empty and already has a few lines of text in it, click the Open button on the top right of that bottom panel, and select New script.

 Step 3: Copy the Indicator Code from GitHub

Since these files are saved as raw text code in this repository, you need to copy them manually:

1. In this GitHub repository, click on the file for the first indicator you want to install: @meyefoundation RSI cheatcode (Candlestick).
2. Once the file opens, look at the top right of the code box and click the Copy raw contents icon (it looks like two overlapping squares).
3. Alternatively, you can highlight all the text in that file and press Ctrl+C (Windows) or Cmd+C (Mac).

 Step 4: Paste the Code into TradingView

1. Go back to your TradingView chart.
2. Delete everything currently inside the Pine Editor panel. It must be completely blank.
3. Click inside the blank panel and press Ctrl+V (Windows) or Cmd+V (Mac) to paste the code you just copied.

 Step 5: Save the Indicator

1. Click the Save button located at the top right of the Pine Editor panel.
2. A pop-up window will appear asking you to name the script.
3. Type in the exact name: @meyefoundation RSI cheatcode (Candlestick) and click Save.

Step 6: Add it to Your Chart

1. Right next to the Save button, click the button that says Add to chart.
2. Look up at your main chart—the indicator is now officially active and running!

---

 Installing the Second Indicator

To install the second indicator, @meyefoundation RSI cheatcode (Trendline), simply repeat the process with a fresh script:

1. In the Pine Editor panel, click Open (top right of the panel) and select New script.
2. Go back to this GitHub repository and open the @meyefoundation RSI cheatcode (Trendline) file.
3. Copy the raw code.
4. Erase any default text in the TradingView Pine Editor, paste the new code, and click Save.
5. Name it @meyefoundation RSI cheatcode (Trendline).
6. Click Add to chart.

---

How to Find Your Indicators in the Future

Once saved, you do not need to paste the code ever again. To find them in the future:

1. Click the Indicators button at the top of your TradingView chart screen.
2. In the menu that pops up, click on Scripts (or My scripts).
3. Click on either @meyefoundation indicator to instantly add it to any chart you are viewing.

---

Troubleshooting Quick-Fixes

* Error Message when saving? Make sure you completely deleted the original placeholder text in the Pine Editor before pasting the GitHub code. If even one character of the old text remains, the code will break.
* Can't see the Pine Editor? Look for a small upward-pointing arrow ($\uparrow$) at the very bottom center/right of your TradingView screen to pull the bottom panel up.
